# MoneyMaster

**为什么开发这个项目？**

1. 桌面端找不到好看好用的记账软件
2. 找不到可以支持导入微信支付宝账单数据的记账软件
3. 学习SQL语法、PyQt5框架、Python数据分析和爬虫

## 项目预期特性

1. 自动导入微信、支付宝账单等数据 
2. 通过数据库本地持久化 
3. 通过PyQt5框架界面可视化 
4. 统计每日、每月、每年的账单数据
5. 折线图、柱状图、饼状图详细分析账单数据
6. 支持通过农历的视角查看账单数据
